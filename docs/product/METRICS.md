# 指标体系

本文档用于定义企业多源知识与数据智能协作平台的产品指标、Agent 指标、RAG 指标、图谱指标和工程指标。

## 1. 产品使用指标

- 日活用户数。
- 周活用户数。
- 单用户平均提问次数。
- 前端会话创建数。
- API 调用次数。
- 人工审批触发次数。
- 图谱 dry-run 次数。
- 图谱正式导入次数。

## 2. Supervisor 路由指标

- 路由准确率：用户问题是否被分发给正确 Agent。
- 路由失败率：路由到错误 Agent 或无法处理的比例。
- 高频规则命中率：确定性规则处理的问题占比。
- LLM 路由兜底占比：需要 LLM 判断的复杂问题占比。
- 重复路由率：同一问题被重复路由到 worker 的比例。

## 3. Agent 执行指标

- `chat` 答案满意度。
- `sqler` 查询成功率。
- `sqler` 写操作审批触发率。
- `coder` 图表生成成功率。
- `graph_kg` 图谱问答准确率。
- `vec_kg` 检索问答准确率。
- 工具调用成功率。
- Agent 端到端任务完成率。

## 4. RAG 检索指标

- Recall@K：正确证据是否被召回。
- Precision@K：召回结果中有效证据比例。
- MRR：正确证据排名是否靠前。
- NDCG：排序质量。
- Answer Faithfulness：答案是否忠于检索证据。
- Answer Relevance：答案是否回应用户问题。
- Context Utilization：答案是否有效利用上下文。

## 5. GraphRAG / 图谱指标

- 实体抽取准确率。
- 关系抽取准确率。
- 实体去重成功率。
- 脏节点比例。
- 图谱问答准确率。
- Cypher 生成成功率。
- 确定性 Cypher 命中率。
- `kg_eval.py` 回归命中率。

## 6. HITL 安全指标

- 危险操作识别准确率。
- 审批触发召回率。
- 误拦截率。
- 审批通过率。
- 审批拒绝率。
- 审批后恢复成功率。
- 审批日志完整率。

## 7. 记忆指标

- 短期 checkpoint 恢复成功率。
- 长期记忆召回准确率。
- 记忆污染率。
- 记忆更新成功率。
- 记忆删除成功率。
- 用户记忆满意度。

## 8. 工程性能指标

- API 平均响应时间。
- P95 响应时间。
- 首 token 延迟（加入流式输出后）。
- 单次请求模型调用次数。
- 单次请求 token 成本。
- Neo4j 查询延迟。
- Milvus 查询延迟。
- 前端构建成功率。
- 后端健康检查成功率。

## 9. A/B 测试指标

后续 RAG 优化应通过 A/B 测试验证：

- 文本切块策略对 Recall@K 的影响。
- 混合检索对 Precision@K 的影响。
- reranker 对 MRR / NDCG 的影响。
- Query rewrite 对答案相关性的影响。
- Multi-query 对召回率和成本的影响。
- HyDE 对复杂问题召回率的影响。

## 10. 核心北极星指标

建议使用以下指标作为阶段性北极星指标：

> 用户能否用自然语言稳定完成跨知识源查询与任务执行。

可量化为：

- 端到端任务完成率。
- 答案正确率。
- 用户满意度。
- 危险操作安全拦截率。
- 回归评测通过率。
